Optional int64

int64: boolean
  • Defined in node_modules/@types/msgpack-lite/index.d.ts:135

It decodes msgpack's int64/uint64 formats with int64-buffer object. int64-buffer is a cutom integer type with 64 bits of precision instead of the built-in IEEE-754 53 bits. See https://github.com/kawanet/int64-buffer

default

false
